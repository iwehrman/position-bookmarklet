1. `$ git clone https://github.com/iwehrman/position-bookmarklet.git`
2. Add a bookmarklet in your browser with the contents of `bookmarklet.js`
2. `$ cd position-bookmarklet`
3. `$ sudo npm install -g serf`
4. `$ serf`
5. Navigate to an HTTP web page and click the bookmarklet. It should look like this: !["pos"](http://cloud.wehrman.org/image/2v422i3G112r/Screen%20Shot%202014-06-05%20at%2010.29.04%20AM.png "pos")
